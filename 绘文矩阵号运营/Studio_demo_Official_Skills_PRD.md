## 1. 小红书图文模板仿写

**Skill ID**: 1
**描述**: 上传品牌文案与图片模板，AI 自动解构版式与配文区域，按主题批量生成同风格图文。
**适用平台**: 小红书
**交付物**: 
- 封面图（N 张）+ 标题 + 正文
- 每张配图含 AI 按主题实时生成的配文

### 1.1 全局设置 (Global Settings)
这部分参数在一次任务中对所有生成内容共用共享。
- **配图模板 (必填)**: 用户上传品牌图文模板图片组 (PNG/JPG) 或 ZIP。AI 会自动解构版式。并且自动锁定“每篇配图张数”。
- **每篇配图张数**: 允许用户指定生成几张图选项（被模板锁定后不可改）。
- **正文结构参考 (选填)**: 用户粘贴参考文章，AI 将仿其结构和语气生成正文。

### 1.2 逐行参数 (Row Parameters)
这部分参数每篇生成内容不同，在单次试用中以表单形式展示，在批量任务中体现为数据表格的列。
- **创作主题与要点 (必填)**: 第一行填主题名称（如车型名、产品名），后续行填核心卖点，合并在同一字段填写。
- **图片图源 (选填)**: 本主题对应的图集链接 / CDN，每行可填写不同图源，不填则 AI 生图。

## 4. 单一平台发布类 Skill (Publishing Skills)

基于各个平台发布配置项的差异性，发布动作被拆分为针对每个平台的**原子 Skill**。用户既可以单次调用这些 Skill 执行存量内容的发布，也可以在“组合工作流”中将它们与生成类 Skill 串联使用。

### 4.1 发布至小红书

**Skill ID**: 41
**描述**: 将存量内容一键分发至指定的小红书账号，在批量任务中支持从内容库选择或 Excel 导入多篇内容进行大并发发布。
**适用平台**: 小红书

#### 交互流（批量发布任务）
批量发布能力在工作台中具有专属的交互流程设计，重点解决“账号-内容一对一映射”及“个性化发布配置”问题：

1. **选择内容素材 (Step 2)**
   - **内容库调取**: 提供卡片式的内容网格，支持关键词和排序方式筛选。每张卡片含题图缩略、文章标题、更新时间和类型。
   - **Excel 导入**: 支持直接调起批量导入功能。
   - **状态统计**: 底部显示已选取素材数量。

2. **配置分发并预览 (Step 3)**
   - **账号组快速分配**: 顶部工具栏支持下拉选择“账号组”（如矩阵号组、教育赛道号组），选中后系统自动将所选文章按照该账号组内部顺序进行分配映射（自动匹配填入表格）。
   - **发布配置预览表格**: 
     - 包含列：「创作文章」、「发布账号」、「发布配置」、「操作」。
     - **创作文章列**: 左侧展示缩略图、标题和原文章属性状态。
     - **发布账号列**: 用户可在行内单独调整绑定的账号，支持展示账号状态（若账号掉线会有橙色提醒引导绑定，若未分配则保留待分配提示）。
     - **发布配置列**: 显示核心参数快照摘要（封面图截面、标题、IP 及发布状态异常警告）。
   - **配置详情抽屉 (Config Drawer)**: 点击表格内的“展开配置”，右侧会滑出该行的完整小红书配置参数抽屉面板。
     - 抽屉顶部展示当前编辑的平台和账号名称，包含快捷导航左右箭头，可无缝切换上下篇目而无需关闭抽屉面板。
     - **可配置的小红书原生参数**：
       - **内容类型切换**: 「图文」与「千字长文」。
       - **展示封面**: 支持最多18张图网格拖拽排序（实时数字预览如 5/18）。
       - **标题与话题**: 支持标题字段及话题组件插入，话题具有字数合规提醒。
       - **地理/合集与社交**: 可添加地点、合集名称、关联互动群聊卡片。
       - **权限与合规声明**: 原创声明勾选框、内容类型预警文本声明、可见范围控制（公开/仅互关/仅自己）。
       - **互动开关**: 允许合拍、允许复制正文。
       - **规划管控**: 发文位置 IP 定位机制（当前区域自动获取）、发布排期（立即/定时发布）。
   - **审核策略**: 表格下方单独提供任务总体审核规则设定——「AI 自动预审」（智能过滤高危敏感词/不推荐词，并在草稿箱自动打回）或「全人工审核」（进入待审核状态要求人工检核通过后方可执行推送）。

## 5. 生成+发布组合类 Skill

### 5.1 小红书一键模板创作+发布

**Skill ID**: 51
**类型**: 生成+发布型
**描述**: 上传品牌模板，按主题批量生成同风格图文，并直接分发至指定小红书账号，实现从内容创作到发布上线的全流程自动化。
**适用平台**: 小红书
**has_publish_capability**: true
**核心特性**: 规划一次、自动运行、后台执行，用户仅在审核标记或发布失败时介入
**交付物**:
- 封面图（N 张）+ 标题 + 正文（已发布至对应账号）

#### 5.1.1 全局设置 (Global Settings)

生成参数（在一次任务中对所有生成内容共用）：

- **配图模板 (必填)**: 用户上传品牌图文模板图片组 (PNG/JPG) 或 ZIP。AI 自动解构版式并锁定"每篇配图张数"。
- **每篇配图张数**: 允许用户指定生成几张图选项（被模板锁定后不可改）。
- **正文结构参考 (选填)**: 用户粘贴参考文章，AI 将仿其结构和语气生成正文。

审核策略（影响后台执行流程）：

- **AI 自动预审**: 每篇生成后立即预审，通过则直接进发布队列；标记则通知用户处理。
- **全人工审核**: 全部生成完毕后进待审核状态，用户批量审核后再发布。

**注**: 发布配置（内容类型、话题、地点、可见范围、发布时间等）**跟随账号默认设置**，不在全局设置中配置。

#### 5.1.2 逐行参数 (Row Parameters)

每篇生成内容不同，在批量任务中体现为数据表格的列：

- **创作主题与要点 (必填)**: 第一行填主题名称（如车型名、产品名），后续行填核心卖点，合并在同一字段填写。
- **图片图源 (选填)**: 本主题对应的图集链接 / CDN，每行可填写不同图源，不填则 AI 生图。
- **发布账号 (必填)**: 目标发布账号。选择后自动加载该账号的默认发布配置。

**账号配置来源优先级**:
1. 账号管理中的预设默认参数
2. 历史任务中该账号最近一次使用的配置

可在逐行的「展开配置」抽屉中查看或手动调整所有小红书发布参数。

#### 5.1.3 批量任务配置流程

**Step 1: 选择 Skill**
选择「小红书一键模板创作+发布」。

**Step 2: 全局设置**
- **生成参数**: 配图模板、配图张数、正文结构参考。
- **审核策略**: AI 自动预审 / 全人工审核。

**Step 3: 数据表格**

表格列：创作主题与要点 | 图片图源（选填） | 发布账号 | 配置状态 | 操作

**关键功能**:
- **账号组快速分配**: 支持下拉选择账号组，系统按组内顺序自动映射至各行。
- **同一账号可出现多次**: 同账号在多行重复出现允许，发布间隔由系统自动处理。
- **配置状态指示**:
  - ✓ 已就绪：该账号已有默认配置。
  - ⚠ 待配置：该账号无默认配置，需用户手动填写。
- **展开配置抽屉**: 点击「展开配置」查看或调整该行的完整小红书发布参数：
  - 内容类型（图文 / 千字长文）
  - 封面图选择与排序（最多 18 张）
  - 标题 + 话题
  - 地点 + 合集 + 社交关联
  - 权限与合规（原创声明、可见范围）
  - 互动开关（允许合拍、允许复制）
  - IP 定位 + 发布排期（立即 / 定时）

#### 5.1.4 后台执行流程 (Autopilot)

提交后，系统**全自动运行**以下流程，用户无需干预：

```
生成中（流式，每篇完成立即预审）
        ↓
[AI 自动预审]                [全人工审核]
通过 → 直接进发布队列        全部 → 待审核
标记 → 通知用户处理          用户批量审核后 → 发布队列
        ↓
发布中（按账号配置执行）
        ↓
已完成（内容入内容库，打「矩阵批次」标签）
```

**用户介入时机**:
- **AI 标记内容**: 在批量任务中的待审核内容卡片处理（通过 / 编辑 / 丢弃）。
- **发布失败**: 系统通知，用户决定是否重试失败项。

#### 5.1.5 试用流程 (Single Run Trial)

由于本 Skill 具备发布能力（has_publish_capability: true），试用流程分为**两个阶段**：

**Phase 1 — 配置与生成**
右侧面板包含：
- **全局设置区**: 配图模板、正文结构参考。
- **单篇参数区**: 创作主题与要点（主题+卖点合并填写）、图片图源（选填）。
- **发布账号**: 选择账号（自动加载默认配置，可展开查看 / 编辑）。
- 点击「生成并预览」。

**Phase 2 — 预览与确认**
展示生成结果：
- 封面图组、标题、正文
- AI 预审结果（✓ 通过 / ⚠ 标记）

操作选项：
- **[重新生成]**: 返回 Phase 1 调整参数重试。
- **[确认发布]**: 发布本篇内容至所选账号，内容入库。
- **[⚡ 批量使用 →]**: 进入新建批量任务弹窗（干净进入，不携带试用参数，但 Skill 已自动定位到本 Skill）。
